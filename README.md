# 🧠 QuizApp

A simple and interactive quiz application built with **HTML**, **CSS**, **JavaScript**, and **Bootstrap**.

## 📋 Features

* Multiple-choice questions
* Four answer options per question
* Instant answer validation
* Progress bar to track quiz completion
* Final score display
* Restart quiz functionality

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/QuizApp.git
```

2. Navigate to the project directory:

```bash
cd QuizApp
```

3. Open `index.html` in your preferred web browser.

## 🎮 How to Use

1. Launch the application in your browser.
2. Read the displayed question.
3. Select one of the available answers.
4. Click **"Next Question"** to continue.
5. After completing all questions, your final score will be displayed.
6. Click **"Play Again"** to restart the quiz.
